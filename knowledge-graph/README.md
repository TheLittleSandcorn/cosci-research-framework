# CoSci Knowledge Graph

Minimal machine-readable knowledge structure for Phase 1.

## Purpose

This directory contains the foundational semantic structures of the CoSci framework.

Current scope:

- semantic boundaries
- validation rules
- ontology model
- RDF graph generation
- graph validation

## Current Status

Phase 1:
- Markdown → RDF transformation
- Minimal ontology
- Type-scoped validation
- Semantic boundary enforcement

Not implemented yet:

- inference
- SHACL validation
- graph reasoning
- drift metrics
- CI/CD enforcement

## Discipline Principles

- Explicit semantics over implicit assumptions
- Minimal ontology before expansion
- Validation before abstraction
- Stable structure before automation
- Reduction over semantic inflation
